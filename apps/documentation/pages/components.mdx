# Web Components

Web Components for _your_ federated cloud infrastructures, our suite of web components are designed to be used in any web application, and are built to be
interoperable with any cloud infrastructure that implements the GA4GH standards. _Got a different implementation?_ No problem, head to out [UI solution](/ui) and
cook your own web components.

## Goals

<Steps>

### Interact with _your_ TES, WES and more

User friendly interface for _your_ TES, WES and more. A GUI for _your_ federated
cloud infrastructure.

### Extensible

Allow to swap and replace various loosely coupled features
of this component to support other/custom TES implementation.

### Styling

Allow you to style the component based on you design language and
brand identity, non programmatically! via `CSS parts` and `CSS variable`.
